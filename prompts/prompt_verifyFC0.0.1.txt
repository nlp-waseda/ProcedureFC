Each node represents a step, and if an entity obtained from a step is required in a subsequent step, that entity is included as the label of the edge (arrow). # Guidelines for Judgment

The basic policy for judgment is as follows: if the evidential document contains information that explicitly contradicts the content of the flowchart under evaluation, then the flowchart should be judged as false. Otherwise, the flowchart should be determined to be true. You must find a statement in the evidence that explicitly denies the description, cite it in the reasoning section, and make your judgment accordingly.
